# Emergency Command Dashboard

A professional emergency command dashboard UI built with React and Mapbox.

## Features

- Dark theme control-room style interface
- Full-screen layout with 70/30 split
- Interactive Mapbox map with unit markers
- Command panel with incidents, unit status, and system metrics
- Minimal, serious design focused on clarity

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Note

This app uses a public Mapbox token for demonstration. For production use, replace the token in `src/components/MapView.jsx` with your own Mapbox access token.
